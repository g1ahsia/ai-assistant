安裝說明 Installation Instructions

macOS 安裝步驟 macOS Installation Steps

  1. 下載 Panlo-macOS.dmg 檔案 Download the Panlo-macOS.dmg file
  2. 開啟下載的 .dmg 檔案 Open the downloaded .dmg file
  3. 將 Panlo 應用程式拖曳到「應用程式」資料夾 Drag the Panlo application to the Applications folder
  4. 從「應用程式」資料夾或 Launchpad 開啟 Panlo Open Panlo from the Applications folder or Launchpad
  5. 首次開啟時，可能需要在「系統偏好設定」中允許來自已識別開發者的應用程式 When opening for the first time, you may need to allow apps from identified developers in System Preferences

Windows 安裝步驟 Windows Installation Steps

  1. 下載 Panlo-Windows.exe 檔案 Download the Panlo-Windows.exe file
  2. 開啟下載的 .exe 檔案 Open the downloaded .exe file
  3. 依照安裝精靈的指示完成安裝 Follow the installation wizard instructions to complete the installation
  4. 從桌面捷徑或開始選單開啟 Panlo Open Panlo from the desktop shortcut or Start menu
  5. 首次開啟時，可能需要允許應用程式存取您的檔案 When opening for the first time, you may need to allow the application to access your files
